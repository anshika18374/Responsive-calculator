I. Calculator

This is a simple calculator project built using HTML, CSS, and JavaScript. The calculator can perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

I created this project to improve my understanding of JavaScript, DOM manipulation, and responsive web design.

II. Features

- Addition, subtraction, multiplication, and division
- Dark mode toggle
- Keyboard support
- Backspace button
- Responsive design for mobile devices
- Simple and user-friendly interface
- Error handling for invalid calculations

III. Technologies Used

- HTML
- CSS
- JavaScript

IV. What I Learned

While building this project, I practiced:
- Working with JavaScript functions
- Handling user interactions and events
- Manipulating HTML elements using the DOM
- Creating responsive layouts with CSS
- Implementing dark mode functionality

V. Future Improvements

- Calculation history
- Scientific calculator functions
- Theme customization
- Memory buttons (M+, M-, MR)
